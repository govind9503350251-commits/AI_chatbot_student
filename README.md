# AI Student Chatbot using Ollama

## Project Overview

This project is a simple AI-powered chatbot developed using Python, Ollama, and the Llama 3.2 Large Language Model. It helps students by answering academic questions in natural language while running completely offline.

## Features

- Offline AI chatbot
- Fast response generation
- No API key required
- Secure and private
- Easy to use

## Technologies Used

- Python
- Ollama
- Llama 3.2
- Natural Language Processing (NLP)

## Installation

1. Install Ollama.
2. Pull the Llama 3.2 model:

```bash
ollama pull llama3.2
```

3. Install the Python package:

```bash
pip install ollama
```

4. Run the chatbot:

```bash
python chatbot.py
```

## Project Workflow

1. User enters a question.
2. Python sends the query to Ollama.
3. Llama 3.2 generates the answer.
4. The chatbot displays the response.
